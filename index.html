Repete e Encaixa

Este é um projeto experimental que busca criar composições tipográficas usando programação. Brincamos com as ideias de repetição e encaixe, adotando uma abordagem modular para o grid de composição e tendo algumas opções de movimentar as palavras, que no final podem ser usadas também de forma estática.

Gerar um Layout

Como funciona?

  1. Você digita uma frase

    Só colocar qualquer frase que desejar!

  2. Seleciona as cores

    Mude as cores dos elementos como preferir.

  3. Seleciona o movimento

    Decida a forma que as palavras irão se movimentar.

  4. E pronto! Só baixar o Layout gerado.

    Você tem a opção de baixar uma imagem do seu Layout.
