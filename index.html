Repete e Encaixa

"Repete e Encaixa" é um projeto experimental que visa criar composições tipográficas por meio de programação. O projeto explorou os conceitos de repetição e encaixe, utilizando uma abordagem modular para o grid de composição.

Gerar um layout

Como funciona?

  1. Você digita uma frase;

  2. Seleciona as cores;

  3. Seleciona o movimento;

  4. E pronto! Só baixar o layout gerado.
